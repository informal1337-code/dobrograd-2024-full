# Установка MySQL:
1. Скачай последнюю версию MariaDB по **[ссылке](https://mariadb.com/downloads/)**
2. Запусти установщик.
3. Тебе предложат выбрать компоненты, которые нужно установить. В рамках разработки достаточно будет конфигурации на скриншоте ниже. Обрати внимание, что помимо сервера мы также устанавливаем HeidiSQL. Это программа для взаимодействия с сервером, она нам понадобится в будущем.
https://i.imgur.com/yN64ieC.png
4. На следующем шаге тебе предложат указать пароль от главного аккаунта сервера (имеет полный доступ). В рамках работы с локальной базой данных рекомендуется этого не делать. Сервер будет использоваться только тобой, и если ты не будешь указывать пароль, не придется ничего настраивать дополнительно. Не забудь также поставить галочку напротив «Use UTF8 as default server’s character set»:
https://i.imgur.com/0GJAk0P.png
5. На следующем шаге чаще всего менять ничего не нужно:
https://i.imgur.com/DbUFw2v.png
6. Наконец, дождись окончания установки. После этого установщик можно закрыть.
# Подключение к ДБ
После установки сервера у тебя должен был установиться HeidiSQL - интерфейс для взаимодействия с базами данных. Если этого не произошло, скачай и установи ее по **[ссылке](https://www.heidisql.com/download.php)**
https://i.imgur.com/WQDksXC.png

Запусти ее.

В открытой программе нажми «New»:
https://i.imgur.com/hLweTHK.png

В разделе «Settings» укажи следующие параметры:

Network type:	MariaDB or MySQL (TCP/IP)
Hostname / IP:	127.0.0.1
User:	root
Password:	(ничего)
Port:	3306

https://i.imgur.com/X4CGfeK.png

Затем сохрани (Save) и выполни вход (Open).

Если хочешь, можешь удалить базу данных «test». Для этого нажми правой кнопкой мыши по ней в списке и выбери «Drop».

Все таблицы должны содержаться в конкретной базе данных.

Дальше, создаешь новую ДБ
https://i.imgur.com/jIBYdyO.png
и называешь ее dbg (никак иначе.)

# все, ты сделал самое главное - установил базу данных, и теперь можешь игратся на локалке с друзьями (STEAM CMD ONLY!!!!)

# ELSE ты на хостинге, буду показывать пример на моем. 
1. заходишь на хостинге в вкладку Databases (или что то подобное)
2. Нажимаешь на New database после чего высветится меню "Create new database", там указываешь в Database Name: dbg а в Connections From оставляй пустоту.
3. нажимаешь на create new database.
4. теперь тебе нужно получить и записать все данные бд (пароль, имя, и тд.) и зайти в файл _config\lua\config\octolib_sv.lua и там будет:
CFG.db = {
	host = 'localhost',
	user = 'root',
	pass = '',
	port = '3306',

	main = 'dbg',
	admin = 'dbg',
	shop = 'dbg',
}

заполняешь так:
CFG.db = {
	host = 'хост, --пример db.worldhosts.fun',
	user = 'юзер, --примерно так выглядит он везде: u17670_J7m4qk',
	pass = 'пароль, --ну тут понятно',
	port = '3306', -- оставляем.

	main = 'dbg', -- меняем на тот нейм который выдал нам хост, пример: s17675_dbg
	admin = 'dbg', -- меняем на тот нейм который выдал нам хост, пример: s17675_dbg
	shop = 'dbg', -- меняем на тот нейм который выдал нам хост, пример: s17675_dbg
}

Все, если ты увидел в консоли Database connected то у тебя все получилось!
